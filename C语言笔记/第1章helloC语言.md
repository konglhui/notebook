# 第一章笔记

## hello word第一个程序

```
# include <stdio.h>
int main()
{
	printf("hello word\n");
	return 0
}
```

以#开头的语句称为预处理器指令

<stdio.h>是standard(标准) input output . header（头，头文件）

void 是返回值

main() 函数是C程序处理的起点或入口。

;代表语句的结束
