# 第一章作业
## 1.1

假设有一个包含128个名字的有序列表，你要使用二分查找在其中查找一个名字，请问最多需要几步才能找到？ 

$log_{2}128 = 7$所以7步

## 1.2 

上面列表的长度翻倍后，最多需要几步？

$log_{2}(128\times 2) = 8$所以8步
